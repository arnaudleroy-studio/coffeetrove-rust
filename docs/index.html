Rust crate for CoffeeTrove coffee data platform. Async access to 440K+ cafes and brewing guides.

About CoffeeTrove

This library provides idiomatic Rust bindings for CoffeeTrove, the world's most comprehensive coffee data platform. Search 440,000+ cafes across 80+ countries, explore brewing methods, and access the Golden Drop scoring system.

CoffeeTrove combines cafe discovery with deep coffee knowledge -- from bean origins and roasting profiles to detailed brewing guides with precise ratios and temperatures. The Golden Drop score provides a standardized quality metric across every cafe in the directory.

# Installation

Rust 1.70+ (2021 edition) 
# Add to Cargo.toml
[dependencies]
coffeetrove = "0.1"
tokio = { version = "1", features = ["full"] }

# Or via cargo
cargo add coffeetrove

# Quick Start

use coffeetrove::{Client, CafeSearch};

#[tokio::main]
async fn main() -> Result<(), Box<dyn std::error::Error>> {
    let client = Client::default();

    // Search 440K+ cafes worldwide
    let cafes = client
        .search_cafes(CafeSearch {
            query: "Portland".into(),
            min_score: Some(40),
            limit: Some(10),
            ..Default::default()
        })
        .await?;

    for cafe in &cafes {
        println!(
            "{} - Golden Drop: {}",
            cafe.name, cafe.golden_drop_score
        );
    }

    // Get brewing method details
    let method = client.get_brewing_method("french-press").await?;
    println!("Brew time: {}", method.brew_time);
    Ok(())
}

# Features

  • Fully async with tokio, zero-copy deserialization
  • 440,000+ cafe directory
  • Golden Drop scoring system
  • Brewing method guides
  • Coffee bean database

# API Reference

ASYNC client.search_cafes(params: CafeSearch) -> Result<Vec<Cafe>>

Search the global cafe directory with filters for score, city, and chain type.

ASYNC client.get_brewing_method(slug: &str) -> Result<BrewingMethod>

Detailed brewing guide with ratios, grind size, water temperature, and timing.

ASYNC client.list_beans(opts: BeanQuery) -> Result<Vec<Bean>>

Browse coffee bean varieties with tasting notes and origin information.

ASYNC client.get_origins() -> Result<Vec<Origin>>

Get all coffee-producing regions with altitude, harvest seasons, and flavor profiles.
